Rusoto

Rusoto is an SDK for Amazon Web Services. It follows best practices for AWS Credentials.

The library is designed with use on AWS EC2 instances in mind but can be used as an AWS CLI replacement.

Credentials

Credentials are sourced from environment variables, AWS credentials profile and IAM instance profiles, in that order. IAM instance profile credentials are refreshed automatically as needed.

Supported services

  • SQS
  • S3 (partial implementation)

Requests and request signing

Rusoto uses AWS Signature 4 to sign requests.
